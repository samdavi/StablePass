About

This password Generator has been created as part of a final year project at Bournemouth University for Cyber Security Management Student - Samuel Davies.  

This tool is used to benefit society in the way that it creates passwords that are secure by three core values:

  • Complex Passwords: Combination of upper, lower, numerical and special characters.
  • Random Passwords: Passwords that are not known gramatically or within a dictionary.
  • Database Leak Protection: Passwords are generated if the plaintext for a password hash cannot be easily found online.
